🧠 Dopamine Study

«Exploring motivation, rewards, and engagement through interactive learning experiences.»

Dopamine Study is an interactive web application that explores how reward systems, instant feedback, and challenge-based progression influence user engagement and motivation.

The project combines competitive gameplay mechanics with SAT-style cognitive questions to demonstrate concepts from cognitive psychology and human-centered design, showing how digital experiences can encourage focus, learning, and continued participation.

---

✨ Features

- 🎮 Challenge-Based Gameplay
  
  - Interactive challenges designed around progression and engagement.

- 🧩 SAT-Style Cognitive Questions
  
  - Practice problem-solving through structured cognitive tasks.

- ⚡ Instant Feedback System
  
  - Receive immediate results and performance feedback.

- 🏆 Reward-Driven Progression
  
  - Experience game mechanics designed to encourage motivation.

- 📊 Performance Tracking
  
  - Monitor progress and improvement over time.

- 📱 Responsive Interface
  
  - Clean and accessible design across devices.

---

🎯 Objective

Dopamine Study was created to explore how game mechanics and reward loops influence user motivation, attention, and engagement.

The project takes inspiration from concepts such as:

- Reward prediction
- Reinforcement mechanisms
- Attention management
- Habit formation
- User-centered design principles

---

🧠 Educational Value

This project demonstrates how psychological concepts can be applied to digital product design.

It serves as an interactive exploration of how elements like challenges, feedback, and rewards can shape user experiences and learning environments.

Note: This project is an educational and experimental application. It is not intended to represent clinical research or validated psychological findings.

🚀 Getting Started

Prerequisites

- Node.js (LTS recommended)
- npm / yarn / pnpm

Installation

git clone <repository-url>

cd dopamine-study

npm install

Run Development Server

npm run dev

Open the application:

http://localhost:3000
